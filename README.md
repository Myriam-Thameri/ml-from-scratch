# Machine Learning From Scratch

Implementations of Machine Learning algorithms from scratch using **Python and NumPy**.
The goal of this repository is to deeply understand how machine learning algorithms work internally without relying on high-level libraries such as scikit-learn or TensorFlow.

This project focuses on learning the **mathematics, optimization methods, and algorithmic logic** behind ML models.

---

## Objectives

* Understand the **mathematical foundations** of machine learning
* Implement ML algorithms **without external ML libraries**
* Learn how **training, gradients, and optimization** actually work
* Build an educational resource for anyone learning ML

---

## Implemented Algorithms

### Supervised Learning

* Linear Regression
* Logistic Regression
* K-Nearest Neighbors (KNN)
* Decision Trees
* Support Vector Machines (SVM)

### Unsupervised Learning

* K-Means Clustering
* Principal Component Analysis (PCA)

### Neural Networks

* Perceptron
* Simple Neural Network
* Backpropagation implementation

## Getting Started

Clone the repository:

```bash
git clone https://github.com/Myriam-Thameri/ml-from-scratch.git
cd ml-from-scratch
```

Install dependencies:

```bash
pip install numpy matplotlib
```

Run an example:

```bash
python linear_regression/linear_regression.py
```
